# NovaTech — Proyecto Final E-Commerce

Tienda online hecha con React + Vite + React Router + Material UI +
Context API, usando la API de [Platzi](https://fakeapi.platzi.com/).

## Cómo correrlo

```bash
npm install
npm run dev
```

## Stack

- React 18
- Vite
- React Router DOM v6
- Material UI v5
- Context API (CartContext, UserContext)
- Custom Hooks (useProducts, useCategories)
- Fetch API nativa

Para una explicación archivo por archivo, ver [`EXPLICACION.md`](./EXPLICACION.md).
